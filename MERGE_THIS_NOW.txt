════════════════════════════════════════════════════════════════

🔴 BUG #1: WRONG ROUTER ADDRESS (Line 387)
─────────────────────────────────────────────────────────────
BEFORE:  const routerAddress = ROUTER_ADDRESSES[selectedChainId];
         ^^^^^^^^^^^^^^^^^^^^^^ Points to FeeRouter!

AFTER:   const liquidityRouterAddress = LIQUIDITY_ROUTER_ADDRESSES[selectedChainId];
         ^^^^^^^^^^^^^^^^^^^^^^^^^^^^ Points to LiquidityRouter!

Impact: Code was calling LiquidityRouter functions on the wrong contract!

════════════════════════════════════════════════════════════════

🔴 BUG #2: WRONG FUNCTION CALL (Lines 397-421)
─────────────────────────────────────────────────────────────
BEFORE:  await approve({         ← Wrong! This is for ERC20 approvals
           functionName: "swapExactInputUniswapV3",
         });

AFTER:   await writeContractGeneric({  ← Correct! Executes transaction
           functionName: "swapExactInputUniswapV3",
         });

Impact: Swap wasn't being executed as a transaction!

════════════════════════════════════════════════════════════════

🔴 BUG #3: MISSING ETH VALUE PARAMETER (Lines 397-421)
─────────────────────────────────────────────────────────────
BEFORE:  await writeContractGeneric({
           args: [...],
           // NO value parameter! ← Contract got 0 ETH
         });

AFTER:   await writeContractGeneric({
           args: [...],
           value: isNativeETH ? amountIn : undefined, ← Sends ETH!
         });

Impact: Contract expected ETH but received 0 → EXECUTION REVERTED!
