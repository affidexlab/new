# DECAFLOW WEEK 1 - DETAILED GRAPHICS GENERATION PROMPTS

## Project Overview
**Brand:** DecaFlow (DeFi Protocol)  
**Campaign:** 2-Week Content Calendar - Week 1  
**Total Graphics Needed:** 7 (One per day)  
**Platform:** Twitter/X (Primary), with multi-platform compatibility  

---

## Global Design Specifications

### Technical Requirements
- **Dimensions:** 1200 x 675 pixels (16:9 aspect ratio)
- **Resolution:** 72 DPI (web optimized)
- **File Format:** PNG-24 (with transparency support)
- **Color Mode:** RGB
- **Max File Size:** <5MB for optimal Twitter loading

### Brand Color Palette
```
PRIMARY COLORS:
- Electric Blue: #00D4FF (RGB: 0, 212, 255)
- Deep Purple: #8B5CF6 (RGB: 139, 92, 246)
- Cyber Violet: #A855F7 (RGB: 168, 85, 247)

ACCENT COLORS:
- Warning Orange: #FF8C00 (RGB: 255, 140, 0)
- Gold: #FFD700 (RGB: 255, 215, 0)
- Success Green: #10B981 (RGB: 16, 185, 129)
- Alert Red: #EF4444 (RGB: 239, 68, 68)

BACKGROUND COLORS:
- Darkest: #0A0E1A (RGB: 10, 14, 26)
- Dark Secondary: #1A1F35 (RGB: 26, 31, 53)
- Dark Tertiary: #2D3250 (RGB: 45, 50, 80)

TEXT COLORS:
- Primary White: #FFFFFF (RGB: 255, 255, 255)
- Secondary Gray: #E5E7EB (RGB: 229, 231, 235)
- Muted Gray: #9CA3AF (RGB: 156, 163, 175)
```

### Typography Guidelines
```
HEADLINE FONTS (Choose one per graphic):
- Primary: "Space Grotesk" - Bold/ExtraBold (700-800 weight)
- Alternative 1: "Inter" - Bold (700 weight)
- Alternative 2: "Poppins" - Bold/SemiBold (600-700 weight)

BODY/SUPPORTING TEXT:
- Primary: "Inter" - Regular/Medium (400-500 weight)
- Alternative: "Roboto" - Regular (400 weight)

CODE/MONOSPACE:
- Primary: "JetBrains Mono" - Regular (400 weight)
- Alternative: "Fira Code" - Regular (400 weight)

SIZE GUIDELINES:
- Main Headline: 48-72pt
- Supporting Headline: 32-48pt
- Body Text: 18-24pt
- Small Text/Labels: 14-16pt
```

### Visual Style Principles
- Modern, sleek DeFi/Web3 aesthetic
- High contrast for mobile readability
- Glassmorphism and gradient effects encouraged
- Subtle glow effects on key elements
- Geometric shapes and angular designs
- Professional yet approachable
- Avoid overly technical or intimidating visuals
