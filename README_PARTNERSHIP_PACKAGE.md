## 🎓 PARTNERSHIP PRINCIPLES

**These DMs are built on:**

1. **Research-First:** Every DM shows you understand their project
2. **Mutual Benefit:** Always frame as win-win
3. **Specificity:** Concrete ideas, not vague "let's partner"
4. **Professionalism:** Respectful but appropriately casual for Web3
5. **Clear CTA:** Always include next step

## 🆘 IF YOU GET STUCK

### **No responses after 1 week?**
→ Check `quick_contact_reference.md` for backup contact methods  
→ Review DMs for possible improvements  
→ Try engaging more on Twitter before DM

### **Got a response but unsure how to proceed?**
→ See "Response Scenarios" in `PARTNERSHIP_OUTREACH_SUMMARY.md`  
→ Use quick response templates in `QUICK_START_GUIDE.md`  
→ Always offer to schedule a call (don't negotiate over DM)

### **They're asking for things you don't have?**
→ Be transparent about stage  
→ Offer pilot/trial partnership  
→ Focus on mutual learning opportunity
