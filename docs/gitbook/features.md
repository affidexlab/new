# Core Features

## Cross-Chain Swaps
- Aggregates routes from 0x, Uniswap V3, Aerodrome, and native routers.
- Supports CCIP + CCTP bridging for USDC and other assets.
- Fee: 0.8% platform fee + network gas (Base ~ $0.12 average).

## Privacy Swap
- Powered by CoW Protocol.
- MEV-protected order flow.
- Default on `/app/privacy`.
- Earns 2.5x points per qualified trade.

## Leaderboard
- Weekly + monthly views at `/leaderboard`.
- Ranks traders by points, volume, and transaction count.
- Cash rewards auto-funded by platform fees.

## VDM Staking
- Users stake VDM to earn protocol fee rebates.
- Accessible via `/staking`.
