# Getting Started

DecaFlow is an Arbitrum-native, multi-chain DEX for privacy-preserving swaps, bridges, and liquidity incentives.

## Supported chains
- Arbitrum (primary)
- Ethereum
- Base
- Optimism
- Polygon
- Avalanche

## Quick start
1. Visit **https://decaflow.xyz**
2. Connect a supported wallet (Coinbase Wallet, Rainbow, MetaMask, etc.)
3. Choose the chain + token pair
4. Execute swap or bridge
5. Earn points automatically (2x-7x depending on action)
