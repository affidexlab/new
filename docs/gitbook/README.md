# DecaFlow Documentation

Welcome to the DecaFlow knowledge base. These docs are optimized for GitBook so they can be published at **docs.decaflow.xyz**.  Everything in this folder maps 1:1 to GitBook pages.

### What you can expect here

- **Getting Started** – project overview, supported chains, and quickstart.
- **Features** – deep dives into swaps, privacy, bridges, and staking.
- **Points & Rewards** – how the leaderboard, rewards, and airdrop system work.
- **Quests** – guided missions for new users.
- **Integrations** – Coinbase OnchainKit and future partner guides.

> **Publishing tip:** Import this folder directly into GitBook or connect the repo using GitBook's Git sync feature. Set the custom domain to `docs.decaflow.xyz` from the GitBook workspace settings (refer to GitBook `Settings → Custom Domain`).
