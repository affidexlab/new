# Quests

Quests guide new users through core DecaFlow actions. Each quest awards bonus points on top of the base rates.

| Quest | Reward | Notes |
|-------|--------|-------|
| Make your first swap | +100 pts | Any chain/token combo |
| Trade on 3 chains | +300 pts | Sequential or same session |
| Try privacy swap | +200 pts | Use `/app/privacy` (CoW) |
| Follow @DecaFlowHQ on X | +100 pts | Submit handle for verification |
| Join Telegram community | +100 pts | `https://t.me/decaflowprotocol` |

### Redeeming quests
1. Visit `/quests` inside the DApp.
2. Complete required action (auto-detected for on-chain events).
3. For social quests, submit proof via quest form.
4. Bonus points appear within 5 minutes.

> Automated verification is rolling out. Until then, operations verifies social quests daily using the admin dashboard.
