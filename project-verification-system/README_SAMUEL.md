## 📊 WHAT MAKES SOMEONE ELIGIBLE?

**ELIGIBLE** ✅ if ALL are true:
- Age is 18-35
- NIN Number is present
- NIN Picture is present

**NOT ELIGIBLE** ❌ if ANY of these:
- Age is below 18 or above 35
- NIN Number is missing
- NIN Picture is missing

**Note**: Selfie passport photos are flagged for review but don't automatically disqualify someone.

## 🚨 COMMON QUESTIONS

**Q: What if a row is completely empty?**
A: Mark as NOT ELIGIBLE, Reason: "No data provided"

**Q: What if age is exactly 35?**
A: PASS ✅ (18-35 means 18 to 35 inclusive)

**Q: What if they have NIN number but no picture?**
A: NOT ELIGIBLE (both NIN number AND picture are required)

**Q: What if passport is a selfie?**
A: Still ELIGIBLE (if age + NIN are OK), but FLAG it as "SELFIE" in tracker

**Q: Can't access Google Drive picture links?**
A: Check if you're logged into correct Google account. Ask COO for access.

**Q: Should I verify the NIN numbers are correct?**
A: No, just check if the field is filled. The COO will validate the numbers later.
