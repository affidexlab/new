## 🎨 BRAND GUIDELINES

**Color Palette:**
- Primary: Deep blue (#0E76FD - from your branding)
- Accent: Cyan/Teal (#00D4FF)
- Secondary: Purple gradients (#6B46FF to #B946FF)
- Background: Dark navy/black with subtle gradients
- Text: White/light gray for contrast

**Style:**
- Futuristic and tech-forward
- Clean, professional, not cartoonish
- Gradients and glowing effects
- 3D elements with depth
- Modern sans-serif typography
- High contrast for readability

**Avoid:**
- Overly busy or cluttered designs
- Meme-style graphics (keep professional)
- Stock photo vibes
- Generic crypto clichés

## 💡 PRO TIPS

**1. Consistency is Key**
- Use same color palette across all graphics
- Maintain similar style/aesthetic
- Keep typography consistent
- Create a cohesive visual brand

**2. Text Readability**
- Always high contrast (light text on dark background)
- Avoid placing text over busy areas
- Use text shadows or glows for legibility
- Keep text large enough to read on mobile

**3. Mobile Optimization**
- Remember 80% of Twitter users are on mobile
- Test graphics at phone size
- Ensure key elements visible in thumbnail
- Avoid tiny text or details

**4. Multiple Versions**
- Create both 16:9 (landscape) and 1:1 (square) versions
- Square format shows better in feed on mobile
- Landscape optimal for desktop Twitter
- Test both to see which performs better

**5. Brand Elements**
- Include "DecaFlow" branding consistently
- Use your actual logo if available
- Maintain recognizable visual identity
- Website URL on every graphic
