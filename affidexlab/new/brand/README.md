# DecaFlow Brand Assets

Official brand assets for DecaFlow partner integrations.

## Contents

- `BRAND_GUIDELINES.md` - Complete brand usage guidelines
- `/logos` - Logo files in various formats
- `/badges` - "Powered by DecaFlow" badges
- `/colors` - Color palette files
- `/social` - Social media templates

## Quick Reference

### Logo Files

All logos provided in:
- SVG (vector, scalable)
- PNG (1x, 2x, 3x for different screen densities)

Variations:
- Full logo (icon + wordmark)
- Icon only
- Wordmark only
- Light background version
- Dark background version

### Primary Colors

- **DecaFlow Purple**: `#4F46E5`
- **DecaFlow Blue**: `#0EA5E9`
- **Dark Gray**: `#111827`
- **Light Gray**: `#E5E7EB`

### Typography

- **Primary**: Inter (Google Fonts)
- **Monospace**: JetBrains Mono

### Minimum Sizes

- Full logo: 120px width
- Icon: 32px × 32px
- Badge: 100px width

## Usage

1. Read `BRAND_GUIDELINES.md` for complete guidelines
2. Use provided assets as-is (do not modify)
3. Maintain proper clear space and sizing
4. Get approval for co-branded materials
